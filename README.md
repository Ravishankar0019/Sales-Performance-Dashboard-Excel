# Sales Performance Dashboard

## Project Overview

This project is an interactive Sales Performance Dashboard developed using Microsoft Excel.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Features

- Total Sales KPI
- Total Orders KPI
- Total Quantity KPI
- Highest Sale
- Lowest Sale
- Best Region
- Best Product
- Monthly Sales Trend
- Payment Mode Analysis
- Sales by Category

## Skills Demonstrated

- Data Cleaning
- Data Analysis
- Dashboard Development
- Business Reporting
- Data Visualization
